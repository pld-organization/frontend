# Sahtek Online Front

Application frontend pour Sahtek Online - Plateforme de télémédecine.

## Technologies utilisées

- React 19
- Vite
- React Router
- Axios
- React Icons

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

L'application sera disponible sur http://localhost:5173

## Build pour production

```bash
npm run build
```

## Déploiement

L'application est configurée pour utiliser l'API backend déployée sur Render.

## Fonctionnalités

- Authentification (connexion/inscription pour patients et docteurs)
- Tableaux de bord séparés pour patients et docteurs
- Interface responsive

## API Backend

L'application communique avec l'API backend déployée sur : https://authservice-version-90.onrender.com
